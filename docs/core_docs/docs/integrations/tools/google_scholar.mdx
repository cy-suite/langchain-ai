# Google Scholar Tool

The Google Scholar Tool enables your agent to access scholarly articles and publications data through the Google Scholar API. Use it to retrieve article titles, abstracts, authors, and publication information directly from Google Scholar.

## Setup

You will need to get an API key from SerpAPI, which offers a Google Scholar API as part of its services. Sign up on SerpAPI, retrieve your API key from your account settings, and set it as process.env.SERP_API_KEY or pass it in as an apiKey constructor argument.
